# **MUST** have info contact url

The `info` object **MUST** have a `contact:url` property that contains a valid URL to contact the team or person responsible for the API.

## Valid Example

```yaml
info:
  ...
  ...
  contact:
    ...
    url: https://acme.com
    ...
```
